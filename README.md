# ISL-FingerSpell

**Real-Time Indian Sign Language Fingerspelling Recognition System**

A lightweight, webcam-based system that recognizes static Indian Sign Language (ISL) fingerspelling signs in real time using MediaPipe hand landmarks and machine learning вЂ” no GPU, no special hardware required.

## Overview

India has approximately 50 million individuals with hearing and speech impairments (census, 2011) who rely on Indian Sign Language as their primary mode of communication. Existing recognition tools either require specialised hardware, or fail under real-world conditions like varying lighting and different hand shapes.

ISL-FingerSpell addresses this by:
- Extracting 21 hand landmarks per frame using MediaPipe вЂ” invariant to lighting and skin tone
- Classifying signs using a trained Random Forest on 126-dimensional geometric features
- Filtering uncertain predictions below a 70% confidence threshold
- Building words in real time using dwell-based letter confirmation and repeated letter handling
- Evaluating cross-user generalization on a completely held-out test contributor

---

## Recognised Signs

23 static ISL fingerspelling signs:
**A B C D E F G I K L M N O P Q R S T U V W X Z**

Letters H, J, Y are excluded вЂ” they involve hand movement and require sequence modelling (proposed as future work).

## System Pipeline

```
Webcam Feed
    в†“
MediaPipe Hands вЂ” 21 landmarks (x, y, z) per hand
    в†“
126-dimensional feature vector (wrist-relative, both hands)
    в†“
Trained Classifier (Random Forest / MLP)
    в†“
Confidence Filter (в‰Ґ 70% threshold)
    в†“
Smoothing Buffer (15 frames majority vote)
    в†“
Dwell Confirmation (25 frames hold)
    в†“
Word Formation + Display
```

## Installation

**Clone the repository:**
```bash
git clone https://github.com/Dwibon/ISL-FingerSpell.git
cd ISL-FingerSpell
```

**Install dependencies:**
```bash
pip install -r requirements.txt
```

> **Note:** MediaPipe version is pinned to 0.10.9 for `solutions.hands` API compatibility.

---

## Usage

### Run live inference
```bash
python inference/run_inference.py
```

**Controls:**

| Key | Action |
|-----|--------|
| `C` | Clear current word |
| `Q` | Quit |

**On-screen indicators:**

- **Green box** вЂ” hand detected, prediction confident
- **Red / ?** вЂ” no hand detected or below confidence threshold
- **Hold indicator bar** вЂ” shows dwell progress before letter is confirmed

### Collect new training data
```bash
python data-collection/collect_data.py
```

### Retrain models
Open `training/notebooks/train_model.ipynb` in VS Code or Jupyter and run all cells.

## Research

This project is being prepared for submission to micro project which is required for 4th semester, B.Tech. The key contributions are:

1. ISL fingerspelling recognition using pure landmark geometry вЂ” lighting invariant
2. Confidence-aware output filtering (70% threshold) вЂ” no silent misclassification
3. End-to-end word formation with repeated letter handling
4. Cross-user generalisation evaluation on a completely held-out contributor
5. Publicly released ISL fingerspelling landmark dataset in CSV format

## Limitations

- Static signs only вЂ” dynamic signs (H, J, Y) require sequence modelling
- Single session per user вЂ” performance may vary with extreme lighting
